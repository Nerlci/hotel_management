# hotel_management

## Run dev server

- doc
    1. `cd ./docs/ && npm i`, then `npm run dev`.
- webui
    1. `cd ./shared && npm i`, then `npm run dev`.
    2. In another terminal `cd ./webui && npm i`, then `npm run dev`.
- backend
    1. `cd ./shared && npm i`, then `npm run dev`.
    2. In another terminal `cd ./backend && npm i`, then `npm run dev`.
